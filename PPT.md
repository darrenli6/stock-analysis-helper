## 1. 执行摘要

**达轮·股票分析助手**是一款面向个人投资者与专业机构的 **AI 全栈股票分析平台**，支持 A 股、港股、美股三大市场，通过自然语言交互即可获得覆盖技术面、基本面、宏观环境、资金流向、舆情情报的完整投研报告，并首创 **AI 买卖方多空辩论**功能，让投资决策过程透明、立体。

**核心数据**

| 指标 | 当前数据 |
|------|---------|
| 支持市场 | A股（沪深北）/ 港股 / 美股 |
| 分析维度 | 11个专业模块 |
| 平均生成时长 | < 90 秒完整报告 |
| 技术栈 | Next.js 15 / React 19 / DeepSeek + OpenAI |
| 数据源 | westock-data-clawhub + Tavily 实时新闻 |

**融资诉求：** 本轮寻求战略入驻资源支持，借助 FTC OPC 平台资源加速产品商业化，目标 12 个月内实现付费用户破万。

## 2. 项目背景与愿景

### 2.1 行业痛点

中国有超过 **2.2 亿**个人投资者账户，但专业投研工具长期存在以下问题：

- **信息割裂**：技术指标、财务数据、宏观政策分散在不同平台，需人工整合
- **门槛极高**：专业分析需掌握 KDJ、MACD、筹码分布等复杂概念，散户难以驾驭
- **视角单一**：大多数工具只给出买/卖结论，缺乏多空对立分析，容易被"单向叙事"影响决策
- **更新滞后**：传统研报以日/周为单位，难以响应分钟级的市场变化

### 2.2 我们的回答

> **"用一句话，换一份完整的投研报告。"**

达轮让任何人都能像专业分析师一样提问——"帮我看看腾讯现在适合买入吗"——平台在 90 秒内完成从数据采集、多维建模到最终建议的全流程，并以多空 AI 辩论的形式将不确定性透明化，保护用户独立思考。

### 2.3 愿景

成为**亚太最好用的个人 AI 投研助手**，让普通投资者拥有与机构同等的分析能力。

## 3. 产品与服务

### 3.1 核心分析引擎

用户通过自然语言输入股票名称/代码/代称（"苹果"、"NVDA"、"hk00700"），平台自动完成以下 **7 步工作流**：

```
Step 1  意图解析   → DeepSeek R1 / GPT-4o-mini 识别股票标的
Step 2  标的检索   → 模糊搜索，支持中英文/代码/公司简称
Step 3  数据采集   → 11个并行数据管道同步拉取
Step 4  宏观建模   → 行业景气度 × 公司治理 × 宏观经济
Step 5  技术评分   → 7项买入条件 × 6项高位预警
Step 6  综合建议   → 动态置信度评分 + 止盈止损参数
Step 7  报告落地   → 结构化 Web 报告实时流式渲染
```

### 3.2 十一大分析模块

| 模块 | 内容 | 亮点 |
|------|------|------|
| **技术指标** | MA/MACD/KDJ/RSI/DMI/BOLL/OBV | 量化评分 0-100，7维买入判断 |
| **K线图表** | 180日 K 线 + 成交量 | 可视化趋势回溯 |
| **财务报表** | 利润表/资产负债表/现金流量表 | 多期对比，AI 摘要 |
| **宏观环境** | 行业景气 × 公司治理 × 宏观经济 | Bing 实时检索建模 |
| **筹码分析** | 筹码均价/获利盘/集中度 | 仅 A 股，判断套牢分布 |
| **股东结构** | 十大股东/流通股东/户数变化 | 识别主力动向 |
| **分红数据** | 近5年分红历史 + 实时股息率 | 价值投资参考 |
| **资金流向** | 主力净流入/融资融券/龙虎榜/大宗交易 | 机构行为透视 |
| **公司画像** | 基本信息/行业/管理层/业务简介 | 基本面快览 |
| **资讯情报** | 公司动态/行业分析/竞品/业绩 | Tavily 实时四维搜索 |
| **多空辩论** | AI 扮演多空双方辩手实时博弈 | 全球首创辩论式投资分析 |

### 3.3 多空 AI 辩论（差异化核心功能）

这是达轮区别于所有竞品的旗舰功能。分析完成后，用户可触发 **"买卖方辩论"**：

- 系统让 AI 分别扮演**做多方（Bull）**与**做空方（Bear）**
- 双方基于同一份实时数据进行多轮辩论
- 每一轮辩词均引用具体数据，可验证、可追溯
- 用户在辩论中自行形成判断，而非被动接受单一结论

> 这一功能解决了 AI 分析最大的信任危机——"它只告诉我结果，我不知道为什么"

## 6. 技术架构与核心壁垒

### 6.1 技术栈

```
前端层      Next.js 15 + React 19 + TailwindCSS 4
            · 流式 SSR，极速首屏
            · 实时轮询任务进度，毫秒级状态同步

后端层      Next.js API Routes + PostgreSQL (Drizzle ORM)
            · 异步任务队列，11 个数据管道并行执行
            · 完整任务日志追溯体系

AI 层       DeepSeek V4 Pro（意图识别 + 推理）
            GPT-4o-mini（财务/筹码/股东/分红摘要）
            · 双 AI 引擎互为备份，可用性 99.9%+

数据层      westock-data-clawhub（行情/技术/财报/资金）
            Tavily API（实时新闻四维检索）
            Bing Search（宏观环境实时爬取建模）
```

### 6.2 三大技术护城河

**1. 11 维并行数据采集管道**
所有数据模块同步采集，总耗时 = 最慢单管道耗时，而非累加。这让达轮在 90 秒内完成传统工具需要 30 分钟人工整合的内容。

**2. 宏观环境实时建模**
不依赖静态数据库，而是实时检索 Bing，对行业景气度、公司治理风险、宏观经济信号进行关键词权重量化，动态生成三维宏观系数。

**3. 多空 AI 辩论引擎**
基于同一份实时分析数据，双角色 AI 进行有据可查的辩论，每一条论点均可追溯到原始数据，彻底解决 AI 黑箱问题。
